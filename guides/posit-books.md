# Posit-Affiliated Books — Complete Reference

## Tier 1: Core References (by Hadley Wickham)

### R for Data Science (2nd Edition, 2023)
- **Authors:** Hadley Wickham, Mine Çetinkaya-Rundel, Garrett Grolemund
- **URL:** https://r4ds.hadley.nz/
- **1st Edition:** https://r4ds.had.co.nz/
- **Publisher:** O'Reilly Media
- **Covers:** Import, tidy, transform, visualize, model, communicate
- **Key chapters for style:** Ch 4 (Workflow: code style), Ch 26 (Functions)

### Advanced R (2nd Edition)
- **Author:** Hadley Wickham
- **URL:** https://adv-r.hadley.nz/
- **1st Edition:** http://adv-r.had.co.nz/
- **Solutions:** https://advanced-r-solutions.rbind.io/
- **Covers:** Foundations, functional programming, OOP, metaprogramming, performance
- **Key chapters:** Functionals, Function factories, Tidy evaluation

### R Packages (2nd Edition)
- **Authors:** Hadley Wickham, Jenny Bryan
- **URL:** https://r-pkgs.org/
- **Covers:** Complete package development lifecycle
- **Key chapters:** The whole book is essential for package dev

### ggplot2: Elegant Graphics for Data Analysis (3rd Edition)
- **Authors:** Hadley Wickham, Danielle Navarro, Thomas Lin Pedersen
- **URL:** https://ggplot2-book.org/
- **Publisher:** Springer
- **Covers:** Grammar of graphics, all layers, extensions

### Mastering Shiny
- **Author:** Hadley Wickham
- **URL:** https://mastering-shiny.org/
- **Solutions:** https://mastering-shiny-solutions.org/
- **Publisher:** O'Reilly Media (2021)
- **Covers:** Reactive programming, modules, testing, deployment

### Tidy Design Principles (in progress)
- **Author:** Hadley Wickham
- **URL:** https://design.tidyverse.org/
- **Covers:** API design for R packages following tidy principles

## Tier 2: Domain-Specific (by Posit Authors)

### Tidy Modeling with R
- **Authors:** Max Kuhn, Julia Silge
- **URL:** https://www.tmwr.org/
- **Publisher:** O'Reilly Media (2022)
- **Covers:** Complete tidymodels workflow — recipes, parsnip, workflows, tune

### Feature Engineering and Selection
- **Authors:** Max Kuhn, Kjell Johnson
- **URL:** http://www.feat.engineering/
- **GitHub:** https://github.com/topepo/FES
- **Covers:** Preprocessing, feature creation, selection methods

### Applied Predictive Modeling
- **Authors:** Max Kuhn, Kjell Johnson
- **URL:** http://appliedpredictivemodeling.com/
- **Publisher:** Springer (2013)
- **Note:** Uses caret (predecessor to tidymodels) but concepts are foundational

### Text Mining with R: A Tidy Approach
- **Authors:** Julia Silge, David Robinson
- **URL:** https://www.tidytextmining.com/
- **Covers:** Sentiment analysis, topic modeling, tidy text principles
